# 📧 TechFest Event Email Template

## 📖 About the Project

This project was created as part of the **CodSoft UI/UX Design Internship**.

The objective of this task was to design a modern and engaging event email template that effectively promotes a technology festival while presenting event details in a clear, visually appealing, and easy-to-read format.

## 🎯 Design Objectives

- Attractive event promotion
- Professional email layout

- Clear visual hierarchy

- Mobile-friendly design
- Consistent typography
